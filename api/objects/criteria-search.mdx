Search criteria contains destination, travel dates and the number of pax in each room.

```graphql
type CriteriaSearch {
  checkIn: Date!
  checkOut: Date!
  hotels: [String!]!
  occupancies: [RoomCriteria!]!
  language: Language
  currency: Currency
  nationality: Country
  market: String!
}
```
