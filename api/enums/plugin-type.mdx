```graphql
enum PluginType {
  PRE_STEP
  HOTEL_MAP
  BOARD_MAP
  ROOM_MAP
  CURRENCY_CONVERSION
  MARKUP
  AGGREGATION
  POST_STEP
  COMMISSION
  MARKETS_GROUP
  AMENITY_MAP
  PROMOTION_MAP
}
```

### Values

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>PRE_STEP</b></code>](#)

> PRE_STEP is the first plugin that will be executed, allowing a full range of operations:
> splitting arrays, joining arrays, modifying object values, and adding or removing object instances

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>HOTEL_MAP</b></code>](#)

> HOTEL_MAP allows to match Seller and Buyer hotel codes based on contexts

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>BOARD_MAP</b></code>](#)

> BOARD_MAP allows to match Seller and Buyer board codes based on contexts

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>ROOM_MAP</b></code>](#)

> ROOM_MAP allows to match Seller and Buyer room codes based on contexts

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>CURRENCY_CONVERSION</b></code>](#)

> CURRENCY_CONVERSION allows to apply currency conversion over all the prices in one Option

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>MARKUP</b></code>](#)

> MARKUP allows the application of markup over the price

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>AGGREGATION</b></code>](#)

> AGGREGATION allows aggregating multiple supplier options.

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>POST_STEP</b></code>](#)

> POST_STEP is the last plugin that will be executed, allowing a full range of operations:
> split arrays, join arrays, modify object values, add or remove object instances

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>COMMISSION</b></code>](#)

> COMMISSION plugin allows the user to calculate the commission

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>MARKETS_GROUP</b></code>](#)

> MARKETS_GROUP plugin allows the user to group search result by markets with shared options

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>AMENITY_MAP</b></code>](#)

> AMENITY_MAP allows to match Seller and Buyer amenities codes based on contexts

#### [<code style={{ fontWeight: 'normal' }}>PluginType.<b>PROMOTION_MAP</b></code>](#)

> PROMOTION_MAP allows to match Seller and Buyer promotion codes based on contexts
