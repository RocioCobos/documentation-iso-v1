Indicates the payment method used, which depends on the configured payment supplier.

```graphql
enum GeneratedPaymentMethodType {
  VIRTUAL_CARD
  WALLET
}
```

### Values

#### [<code style={{ fontWeight: 'normal' }}>GeneratedPaymentMethodType.<b>VIRTUAL_CARD</b></code>](#)

> The payment method is a generation of a virtual credit card to send to the supplier.

#### [<code style={{ fontWeight: 'normal' }}>GeneratedPaymentMethodType.<b>WALLET</b></code>](#)

> The payment method is a wallet transaction.
