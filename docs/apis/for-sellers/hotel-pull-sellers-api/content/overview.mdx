# Overview

The Content methods provide access to Buyers to essential static information from Sellers, including details like hotel and destination lists. To effectively manage these methods, familiarize yourself with calls such as `HotelList`, `CategoryList`, `GeographicalDestinationTree`, `RoomList`, `MealPlanList`, and `MetaData`.
