# Overview

You can receive instant API notifications from Travelgate with the `HotelResNotif` message for bookings and the `CancelNotif` message for cancellations. 

Also, it is possible to implement `HotelResRetrieve` messages to obtain a list of bookings from the Inventory Tool System.

import DocCardList from '@theme/DocCardList';
import {useCurrentSidebarCategory} from '@docusaurus/theme-common';

<DocCardList items={useCurrentSidebarCategory().items.filter((item, i) => i > 0)}/>

:::note

Note that card payments are only allowed if you implement `HotelResNotif` message

:::
